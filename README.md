# resource-share

QQ 群资源共享站，使用 VitePress 构建，用于展示电子书资源、BT 种子资源和游戏资源的网址与说明。

## 本地开发

```bash
npm install
npm run docs:dev
```

## 常用命令

```bash
npm run generate
npm test
npm run docs:build
```

- `npm run generate`：根据资源 Markdown 文件生成资源总览和分类索引。
- `npm test`：运行资源索引生成脚本测试。
- `npm run docs:build`：生成索引并构建 VitePress 网站。

## 投稿

群友可以阅读 `docs/contribute.md`，新增资源 Markdown 文件后提交 Pull Request。

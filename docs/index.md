# 资源共享索引

这是一个用于整理电子书资源、P2P 索引资源、游戏资源、创作者内容索引、网络工具和特定内容入口的共享索引。

本站只提供资源网站的网址和说明，真实内容由贡献者通过 Pull Request 补充和维护。

项目仓库：[Matrix-2001/resource-share](https://github.com/Matrix-2001/resource-share)

## 资源分类

- [电子书资源](./resources/ebooks/)：电子书搜索、下载和阅读相关网站。
- [P2P 索引资源](./resources/torrents/)：P2P 文件索引、目录和相关资源站。
- [游戏资源](./resources/games/)：游戏信息、下载和相关工具资源。
- [创作者内容索引](./resources/creator-archives/)：创作者平台内容目录、索引和资料入口。
- [网络工具](./resources/network-tools/)：网络访问、代理服务和相关工具入口。
- [❤大家想看的东西❤](./resources/❤大家想看的东西❤/)：特定内容索引、导航和相关资源入口。

## 参与维护

如果你想补充资源，请阅读 [投稿说明](./contribute.md)，按模板新增资源文件并提交 Pull Request。

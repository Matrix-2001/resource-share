## 简介

uBlock Origin 是一个开源浏览器内容拦截器，项目说明中强调它面向 Chromium 和 Firefox，注重 CPU 与内存效率。

## 特点

- 支持广告、追踪器、弹窗、恶意站点等内容拦截。
- 使用过滤列表和自定义规则，适合需要更细粒度控制的浏览器用户。
- 项目提供文档、发布页和浏览器扩展安装入口。

## 注意事项

浏览器扩展会影响页面加载和站点功能。使用前建议从官方仓库或浏览器扩展商店获取，并留意浏览器平台对扩展机制的支持变化。

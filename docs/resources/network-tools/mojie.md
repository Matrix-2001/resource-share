## 简介

Mojie 是一个网络访问工具相关站点，适合记录代理服务入口、订阅服务说明和相关使用提醒。

## 特点

- 作为网络访问工具入口收录，区别于电子书、BT、游戏和创作者归档类别。
- 适合在详情页记录入口地址、使用说明和状态变化。
- 站点可用性可能随时间变化，需要定期确认。

## 注意事项

网络访问工具可能涉及地区网络政策、账号安全和隐私风险。使用前应了解所在地法律法规，避免提交账号、订阅密钥、邀请链接或其他敏感信息。
